Getting Started


Downloading ripple

Ripple is currently distributed through the buildsupport git module from the Fubu github account. We have plans as early as Friday of this week to remove the submodule and distribute ripple as a ruby gem. If there is enough interest, we might distribute it as a NuGet package for consumption through Chocolately as well.

In the meantime, you can obtain our recent build of ripple at http://build.fubu-project.org.
