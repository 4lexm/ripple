Create Packages

ripple create-packagesCreates the nuget files locally

ripple create-packages[-v, --version ] [-d, --destination ] [-u, --update-dependencies] [-l, --solution ] [-c, --cache ] [-a, --all] [-v, --verbose]
Flags
[-v, --version <version>]Override the version of the nuget file
[-d, --destination <destination>]Specify where the nuget file should be written, otherwise it just goes to the nuget default
[-u, --update-dependencies]Modify the nuspec files to match the current dependency version constraints
[-l, --solution <solution>]override the solution to be cleaned
[-c, --cache <cache>]Override the NuGet cache
[-a, --all]Apply restore to all solutions
[-v, --verbose]Writes all output to the screen
