List

ripple listlists information about the current ripple environment

Usages
lists all projectsripple list [-m, --mode all|published|dependencies|projects|solutions|assemblies] [-n, --nuget <nuget>] [-l, --solution <solution>] [-c, --cache <cache>] [-a, --all] [-v, --verbose]
filters the information shownripple list [-m, --mode all|published|dependencies|projects|solutions|assemblies] [-n, --nuget <nuget>] [-l, --solution <solution>] [-c, --cache <cache>] [-a, --all] [-v, --verbose]
Flags
[-m, --mode all|published|dependencies|projects|solutions|assemblies]Filters the information shown
[-n, --nuget <nuget>]If set, only shows information for a named nuget
[-l, --solution <solution>]override the solution to be cleaned
[-c, --cache <cache>]Override the NuGet cache
[-a, --all]Apply restore to all solutions
[-v, --verbose]Writes all output to the screen
