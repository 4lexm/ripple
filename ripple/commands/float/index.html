Float

ripple floatAllows a nuget to 'float' and be automatically updated from the Update command

ripple float [-m, --min-version ] [-l, --solution ] [-c, --cache ] [-a, --all] [-v, --verbose]

Arguments
nameThe name of the nuget

Flags
[-m, --min-version <minversion>]The minimum version required
[-l, --solution <solution>]override the solution to be cleaned
[-c, --cache <cache>]Override the NuGet cache
[-a, --all]Apply restore to all solutions
[-v, --verbose]Writes all output to the screen
