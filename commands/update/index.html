Update

ripple updateUpdates the nugets for the solution

ripple update[-n, --nuget ] [-p, --preview] [-v, --version ] [-f, --force] [-s, --stability ] [-l, --solution ] [-c, --cache ] [-a, --all] [-v, --verbose]
Flags
[-n, --nuget <nuget>]Only update a specific nuget by name
[-p, --preview]Only show what would be updated
[-v, --version <version>]Version of the nuget
[-f, --force]Forces the update command to override all dependencies even if they are locked
[-s, --stability <stability>]Override the feed-level stability configuration
[-l, --solution <solution>]override the solution to be cleaned
[-c, --cache <cache>]Override the NuGet cache
[-a, --all]Apply restore to all solutions
[-v, --verbose]Writes all output to the screen
